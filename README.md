## 📖 Описание проекта

Страница-дневник с воспоминаниями о каждом спринте обучения. Каждая карточка сопровождается изображением с уникальным CSS-фильтром, передающим эмоциональное состояние.

### Функциональность:
- ✅ 8 карточек с текстами и изображениями
- ✅ Анимированное сердечко (лайк) с разными состояниями
- ✅ Кнопка «Сохранить на память» с модальным окном
- ✅ Адаптивная вёрстка под мобильные устройства (375px)
- ✅ Резиновая шапка с пиксельным шрифтом 

---

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации, медиазапросы)
- JavaScript (модальное окно, лайки)
